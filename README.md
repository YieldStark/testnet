# YieldStark Frontend

## Getting Started

### Prerequisites
- Node.js 18 or higher
- npm or yarn
- Starknet wallet (Argent X or Braavos)

### Installation

1. **Install dependencies**
```bash
npm install
```

2. **Run the development server**
```bash
npm run dev
```

3. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
npm start
```
